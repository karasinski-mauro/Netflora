about=The Netflora Project involves the application of geotechnologies in forest automation and carbon stock mapping in native forest areas in Western Amazonia.
 It is an initiative developed by Embrapa Acre with sponsorship from the JBS Fund for the Amazon.
 This plugin focuses on the Forest Inventory using drones component, using drones and artificial intelligence to automate forest inventory stages for identifying strategic species.
 More than 50,000 hectares of forest areas have already been mapped to compose the Netflora dataset.
 Author: Mauro Alessandro Karasinski.
 Institution: Embrapa Acre.
 Page: https://www.embrapa.br/acre/netflora
 Support: https://fundojbsamazonia.org/
 The plugin uses external ONNX-based models to detect forest species in drone imagery.
 Model weights are not bundled in the plugin package.
 When a model is used for the first time, Netflora can ask the user for permission to download the matching release asset and cache it in the local QGIS profile.
 Python runtimes such as onnxruntime are also external dependencies and must be installed in the QGIS Python environment.
