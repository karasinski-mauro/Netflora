about=Netflora is a QGIS plugin for forest inventory with drones, geotechnologies and artificial intelligence in Western Amazonia.
 It is part of the Netflora Project, developed by Embrapa Acre with sponsorship from the JBS Fund for the Amazon.
 The plugin supports forest inventory workflows with biome-specific detection algorithms, flight-planning tools and optional PDF reporting.
 More than 50,000 hectares of forest areas have already been mapped to help build the Netflora dataset.
 Author: Mauro Alessandro Karasinski.
 Institution: Embrapa Acre.
 Project page: https://www.embrapa.br/acre/netflora
 Support: https://fundojbsamazonia.org/
 External ONNX-based model weights are not bundled in the plugin package.
 When a model is used for the first time, Netflora can download the required release asset and cache it in the local QGIS profile.
 Python runtimes such as onnxruntime must be installed in the QGIS Python environment.
